# Reaction Roles Bot V4

This version fixes npm lock sync issues for Koyeb/Heroku.

### Deployment on Koyeb

1. Add environment variables:
   - `TOKEN`: Your bot token
   - `CLIENT_ID`: Your bot client ID

2. Set build command:
   ```bash
   npm install
   ```

3. Set run command:
   ```bash
   npm start
   ```

### Features
- Reaction roles
- Global commands
- Customizable rotating status
